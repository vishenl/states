# Install Shopify CLI via Homebrew

## Prerequisites Verified

Your system meets all requirements:

- Node.js v22.13.1 ✓
- npm 10.9.2 ✓
- Git 2.46.2 ✓
- Homebrew available ✓

## Installation Steps

### 1. Add Shopify tap to Homebrew

- Run: `brew tap shopify/shopify`

### 2. Install Shopify CLI

- Run: `brew install shopify-cli`

### 3. Verify Installation

- Check version: `shopify version`
- Test help command: `shopify help`
- Confirm CLI is accessible from any directory

## Expected Outcome

- Shopify CLI installed globally
- `shopify` command available in terminal
- Version information displays correctly
- Help menu accessible
